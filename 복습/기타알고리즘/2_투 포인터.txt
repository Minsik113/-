● 투 포인터
: 리스트에 순차적으로 접근해야 할 때 2개의 점의 
위치를 기록하면서 처리하는 알고리즘이다.
=> 음수가 섞여있으면 사용 못한다.

ex) 
문제1. 특정한 합(M)을 가지는 부분 연속 수열 찾기
문제2. 정렬되어 있는 두 리스트의 합집합


● 문제1 구현시 Tip
1. 시작점(start)과 끝점(end)이 첫 번쨰 원소의 인덱스(0)를 가리키게 한다.
2. 현재 부분합이 M과 같다면 카운트한다.
3. 현재 부분합이 M보다 작으면 end += 1
4. 현재 부분합이 M보다 크면 start += 1
5. 모든 경우를 확인할 때까지 2번~4번 과정 반복

● 문제2 구현시 Tip 
-> O(N+M)
1. 정렬된 리스트 A,B를 입력받는다.
2. 리스트 A에서 처리되지 않은 원소 중 가장 작은 원소를 i가 가리키게 한다
3. 리스트 B에서 처리되지 않은 원소 중 가장 작은 원소를 j가 가리키게 한다
4. A[i]와 B[j]중에서 더 작은 원소를 결과리스트에 담는다
5. 리스트 A와 B에서 더 이상 처리할 원소가 없을 때까지 2~4과정을 반복한다.
