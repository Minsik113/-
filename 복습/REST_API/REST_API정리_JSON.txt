● JSON
:데이터를 주고받는 데 사용하는 경량의 데이터 형식
데이터를 주고받는 방식이 개발자마다 다 다르다.
-> 다만, 이렇게 주고받는 데이터의 형식에 대한 표준도 존재한다.
(Java Script Object Notation의 약자)


● JSON데이터
: 키-값 쌍으로 이루어진 데이터 객체를 저장하고 있다.
{
    "id": "G-Dragon"
    , "password": "20201010"
    , "age": 27
    , "hobby": ["programming", "play game(lol)"]
}

: 특정한 형식으로 저장된 데이터에 접근하여 원하는
정보만 찾아서 가공하는 작업
